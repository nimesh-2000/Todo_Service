# Todo Backend 

## 📌 Overview
This is the backend service for the Todo Task Management application.  
It provides RESTful APIs to create, retrieve, and update tasks.

---

## 🚀 Tech Stack
- Node.js
- Express.js
- MySQL

---

## 📂 Features
- Create a new task (title + description)
- Retrieve only the latest 5 pending tasks
- Mark a task as completed (hidden from UI)
- RESTful API design

---

---

# 📗 ✅ Todo Frontend

## 📌 Overview
This is the frontend application for the Todo Task Management system.  
It allows users to create tasks and manage them through a simple UI.

---

## 🚀 Tech Stack
- React.js
- Axios
- CSS

---

## ✨ Features
- Add new tasks (title + description)
- Display latest 5 pending tasks
- Mark tasks as completed
- Responsive split layout UI (Form on left, tasks on right)

---

## 🖥️ UI Layout
- Left side → Task creation form
- Right side → Task list (latest 5 tasks)
